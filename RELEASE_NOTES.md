## VoiceInput v1.0.0

Первый официальный релиз программы голосового ввода для Windows.

### Возможности

- Офлайн распознавание речи (Vosk)
- VAD фильтр - экономия CPU в тишине
- Уведомления Windows и звуковые сигналы
- Выбор микрофона в настройках
- Менеджер моделей - скачивание/удаление
- Статистика использования
- Автозапуск при старте Windows
- Горячая клавиша Ctrl+Shift+S

### Установка

1. Скачайте VoiceInput-Setup-1.0.0.exe
2. Запустите и следуйте инструкциям
3. После установки нажмите Ctrl+Shift+S для начала работы

### Рекомендуется

Скачать большую модель через Настройки - Управление моделями для лучшего качества распознавания.

Размер установщика: ~120 МБ (включает малую модель)
