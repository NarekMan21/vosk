📋 ЧТО НУЖНО:
   ✓ Python 3.8+ установлен
   ✓ Модель Vosk находится в models/vosk-model-small-ru-0.22
   ✓ Все зависимости установлены (pip install -r requirements.txt)

🚀 ШАГИ:

   1. Установите PyInstaller (если еще не установлен):
      pip install pyinstaller

   2. Запустите сборку:
      build.bat
      
      ИЛИ вручную:
      pyinstaller build_exe.spec

   3. Готово! Файл будет в:
      dist\VoiceInput.exe

📦 ЧТО БУДЕТ В EXE:
   ✓ Весь код программы
   ✓ Все зависимости (Python, библиотеки)
   ✓ Модель Vosk (vosk-model-small-ru-0.22)
   ✓ config.json (настройки по умолчанию)

💡 ВАЖНО:
   - Модель должна быть в папке models/ перед сборкой
   - Размер exe будет ~100-200 МБ (включая модель)
   - Exe можно запускать на любом Windows 10/11 без Python!

📤 ДЛЯ ДРУЗЕЙ:
   Просто отправьте им VoiceInput.exe - больше ничего не нужно!
   Они могут запустить его двойным кликом.

❓ ПРОБЛЕМЫ ПРИ СБОРКЕ?

   • "PyInstaller не найден"
     → pip install pyinstaller

   • "Модель не найдена"
     → Убедитесь, что models/vosk-model-small-ru-0.22 существует
     → Скачайте модель с https://alphacephei.com/vosk/models

   • "Ошибка при сборке"
     → Проверьте, что все зависимости установлены
     → Попробуйте: pip install -r requirements.txt --upgrade
