📋 ЧТО НУЖНО:
   ✓ Микрофон подключен и работает
   ✓ Windows 10/11

🚀 ЗАПУСК (2 варианта):

   ВАРИАНТ 1: Если есть VoiceInput.exe
   → Просто дважды кликните на VoiceInput.exe
   → Всё уже внутри, ничего устанавливать не нужно!

   ВАРИАНТ 2: Если есть исходники (папка src)
   → Установите Python 3.8+ с python.org
   → Запустите install.bat (установит зависимости)
   → Запустите run.bat

⌨️ УПРАВЛЕНИЕ:
   Win+H - включить/выключить голосовой ввод
   Ctrl+Shift+O - пауза/возобновление
   Правый клик на иконке в трее (справа внизу) - меню

💡 КАК ИСПОЛЬЗОВАТЬ:
   1. Запустите программу
   2. Нажмите Win+H (или через меню в трее)
   3. Говорите в микрофон - текст появится в активном окне
   4. Голосовые команды:
      - "запятая" → ,
      - "точка" → .
      - "новая строка" → перенос строки
      - "абзац" → двойной перенос

⚙️ НАСТРОЙКИ:
   Правый клик на иконке → "Настройки"
   Можно изменить горячие клавиши и метод ввода

❓ ПРОБЛЕМЫ?

   • Программа не запускается?
     → Убедитесь, что у вас Windows 10/11
     → Попробуйте запустить от имени администратора

   • Не работает микрофон?
     → Проверьте настройки Windows (Параметры → Конфиденциальность → Микрофон)
     → Убедитесь, что микрофон выбран как устройство по умолчанию

   • Текст не вводится?
     → Убедитесь, что активное окно принимает текст (например, текстовый редактор)
     → Проверьте логи в файле voice_input.log

   • Низкая точность распознавания?
     → Говорите четче и не слишком быстро
     → Уменьшите фоновый шум
     → Используйте качественный микрофон
